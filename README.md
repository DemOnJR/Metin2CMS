Metin2CMS
=============
Metin2CMS este un website simplu de folosit destinat serverelor de Metin2.

Contine: 
  - Panou utilizator(schimbare parola, mesaje de la administratori, formular donatii etc.)
  - Magazin de iteme(simplu dar eficient)
  - Panou administrare

Se bazeaza pe:
  - php 5.4+
  - Bootsrap 3
